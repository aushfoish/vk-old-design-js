# Vkontakte.ru

Архивная первая версия точной копии интерфейса ВКонтакте начала-середины 2010-х годов с ограниченным функционалом. 

**Цель проекта:**
* Изучение базовых возможностей веб-платформы (DOM-дерево, нативные браузерные события, императивный подход).

**Изменения:**
* *Интерфейс:*
    - Добавлена демонстрационная навигация по сайту
    - Добавлена полноценная страница с аудиоплеером и стилизованным списком музыки с настройкой громкости воспроизведения и динамической полоской хронометража. 
    - Добавлена имитация раздела "Мои новости" в виде "бесконечной" ленты новостей.
    - удалены лишние пункты в панели навигации

**Планируются в будущем:**
* перенос проекта на React + TS + Zustand, данная версия заброшена.

**Технологии**:
* HTML5 / CSS3 
* Vanilla JavaScript 


# Послесловие

Данный проект я писал самостоятельно на чистом энтузиазме в течение двух месяцев с целью освоить концепцию SPA на базовом уровне. В ходе работы я столкнулся с императивным подходом, который я использовал для реализации нескольких страниц, что в конечном итоге привело меня к пониманию неоспоримых преимуществ и необходимости применения фреймворков - как в отношении оптимизации производительности, так и в отношении нюансов, связанных с пользовательским опытом. Теперь я буду переносить его на React - TS - Zustand.
